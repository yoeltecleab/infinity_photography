<!-- ## Springboot `Photography Session Booking` App -->
 
 ### App highlight features / functionallity  
 
 - Registration and login page for user authenticated bu ***Spring Security***
 - Password encryption using ***BcryptPasswordEncoder***
 - Utilizing CRUD (creating, reading, updating and deleting) operations
 - Tests using ***JUnit*** for app functionality
 - Dynamic pages using ***Thymeleaf***, a template engine
 - Relational Database Management System using MySQL
 
### Process
- Configure the database drive, **username** and **password** in **`application.properties`** located in **`src/main/resources/application.properties`**
- When you run the app, the entity tables are automatically created in MySQL database by default
- However, you have to run the **`query.sql`** located in **`src/main/resources/query.sql`** inorder to insert the data to the **`services`** table. Failure to do so might result in code breakage and **`NullPointerException`**
- The application runs in *port-8080* using **`http://localhost:8080`** If you wish to change the port number, the **`application.properties`** is located in **`src/main/resources/application.properties`** ***However, you must also change the navigation link in the javascript file located in ***`src/main/resources/static/JS/script.js`*** to obtain the full functionality of the application***
