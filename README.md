 ### App highlight features / functionallity  
 
 - Registration and login page for user authenticated bu ***Spring Security***
 - Password encryption using ***BcryptPasswordEncoder***
 - Utilizing CRUD (creating, reading, updating and deleting) operations
 - Tests using ***JUnit*** for app functionality
 - Dynamic pages using ***Thymeleaf***, a template engine
 - 

### Process
- Configure the database drive, **username** and **password** in **`application.properties`** located in **`src/main/resources/application.properties`**
- When you run the app, the entity tables are automatically created in MySQL database. 
- However, you have to run the **`query.sql`** located in **`src/main/resources/query.sql`** inorder to insert the data to the **`services`** table.
  Failure to do so might result in code breakage and **`NullPointerException`**
