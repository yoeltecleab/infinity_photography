An MVC Java Springboot Photography Session Booking App
 
 
 ## App highlight features / functionallity  
 
 - A user can register an account
 - Book for a session of their choice
 - Put payment method and checkout
 
 
## Built With
 - Java 11
 - Spring Framework 5
 - Springboot 2.1.7
 - Maven 3.3
 - MySQL 8.0.15
 - Thymeleaf 3.0.11
 - Lombok 1.18.10
 - Spring Data JPA
 - Hibernate
